# Task Management Dashboard - Demo Script

**Estimated Duration:** 2-3 Minutes

## 1. Introduction (0:00 - 0:30)
- **Visual:** Show the Landing Page / Login Page.
- **Voiceover/Text:** "Welcome to the Task Management Dashboard, a modern, responsive application built with React, Vite, Tailwind CSS, and Zustand. This tool is designed to help you organize your tasks efficiently with a clean and intuitive interface."

## 2. Authentication Flow (0:30 - 0:45)
- **Visual:**
    - Enter a mock email/password (e.g., `demo@example.com`).
    - Click "Sign In".
    - Show success toast notification ("Welcome back!").
- **Voiceover/Text:** "We start by logging in. The app uses a mock authentication system that persists your session, so you stay logged in even after refreshing."

## 3. Dashboard Overview (0:45 - 1:15)
- **Visual:**
    - Land on the Dashboard.
    - Highlight the 3-column Kanban board (Todo, In Progress, Completed).
    - hover over the "Add Task" button.
    - Show the Dark Mode toggle (if available/visible).
- **Voiceover/Text:** "Here is the main dashboard. It features a Kanban-style board divided into 'Todo', 'In Progress', and 'Completed'. The layout is fully responsive, adapting seamlessly from desktop to mobile."

## 4. Task Management (1:15 - 2:00)
- **Visual:**
    - **Create:** Click "Add Task", fill in title/desc, save. Show the new task appearing in 'Todo'.
    - **Edit:** Click a task, change the status to 'In Progress', save. Show it moving columns.
    - **Delete:** Click the delete icon on a task, confirm removal.
    - **Filter:** Type in the search bar to filter tasks by name.
    - **Drag & Drop (if implemented):** Demonstrate dragging a task between columns.
- **Voiceover/Text:** "Managing tasks is effortless. You can create new tasks, edit details, and move them across stages. Real-time search and filtering make it easy to find exactly what you're looking for."

## 5. Technical Highlights (2:00 - 2:30)
- **Visual:** Quickly show the Settings page or code snippets (optional).
- **Voiceover/Text:** "Under the hood, we use Zustand for state management with local storage persistence, ensuring your data is safe. Tailwind CSS provides a polished, consistent design system."

## 6. Conclusion (2:30 - End)
- **Visual:** Log out and return to the Login screen.
- **Voiceover/Text:** "That's a quick tour of the Task Management Dashboard. Simple, fast, and effective. Thanks for watching!"
